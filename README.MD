# Bakuard Collections

---

## Назначение
**Bakuard Collections** это библиотека с реализацией различных стандартных структур данных и алгоритмов. Является
альтернативой Java Collections Framework, добавляя read-only коллекции без нарушения принципа подстановки Барбары Лисков,
единообразный подход к именованию методов и классов, а также более гибкий API для некоторых операций.

---

## Релиз
Минимальная версия Java: 25

Последняя версия: 0.3.2

---

## Ключевые особенности
- Отдельные интерфейсы для readable версий структур данных без мутирующих операций
- Простой способ для конвертации структур из Bakuard Collections в коллекции JCF (через static методы StandardCollections)
- Для удобной конвертации между структурами Bakuard Collections последние содержат конструктор, принимающий Iterable
- Все семантически схожие мутирующие операции, у наследников ReadableLinearStructure, называются похоже или одинаково
- Простая не глубокая иерархия классов
- Реализация специфических структур данных (битовый вектор, циклический буфер)
- Расширенная версия Iterator для линейных структур данных без мутирующих операций
- Отдельный класс для стека, без операций из списка
- Расширенные версии Predicate, Function и Consumer для линейных структур данных включающих индекс элемента
- Более гибкая сигнатура некоторых часто используемых операций в сравнении с JCF
